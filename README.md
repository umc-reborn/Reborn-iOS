# UMC 3기 리본 팀 - Reborn

## 서비스 소개

빵집이나 편의점에서 남은 상품을 폐기하지 않고 이웃들에게 나눠줄 수 있는 중계 IT 플랫폼으로, 가게 사장 화면과 이웃 화면을 분리함으로써 가게 사장은 상품 관리와 리뷰 관리 기능을, 이웃은 상품 보기와 리뷰 작성 기능을 제공
합니다.

## 🖥️ 개발 환경

- Xcode: 15+
- Swift: 5.9+
- 배포타겟: iOS 16+
- 의존성 관리: CocoaPods
- 작업 일정관리: Notion

<br>

## 🔑 핵심 기술 

### 🗃️ 프레임워크
- UI: UIKit
- 암호화: UserDefaults, CoreData
- 카카오로그인: KakaoSDK

### 🗂️ 외부 의존성
- UI: DropDown, Tabman
- 네트워크 통신: Alamofire
